### Code Blocks

```python
def hello_world():
    print("Hello, World!")
    return "This code is syntax highlighted!"

# You can specify the language for proper highlighting
result = hello_world()
```

```javascript
// JavaScript example
const greet = (name) => {
    console.log(`Hello, ${name}!`);
};

greet("Carter");
```
